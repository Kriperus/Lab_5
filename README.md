# Задание 1

### Текст задачи

Найти сумму делителей данного натурального числа.

### Алгоритм решения

1. Для начала необходимо создать саму структуру нашего дерева и его отображения.
Для этого я возьму пример который давали нам на лекции и слегка его видоизменю.
2. Структура бинарного дерева стандартная - слева меньшие значения справа большие,
что мы и имеем теперь в коде.
3. Следующим шагом необходимо заполнить наше дерево. Я сделаю это посредством рандомно
заполняюегося списка с последующим включением в дерево.
4. После заполнения дерева его выводить для чего и пишем функцию printTree.
5. Теперь осталась самая важная задача - деление всех членов и превращение их в
вещественные значения. Изобретем функцию map для дерева которая будет выполнять определенные действия
на всех элементах нашего дерева.
6. Собственно теперь преобразуем типы и запрашиваем у пользователя делитель, после чего выводим
обновленное дерево
7. Теперь остались тесты.

# Задание 2

### Текст задачи

Дерево содержит строки. Найти суммарную длину этих строк

### Алгоритм решения

1. Берем за основу уже имеющееся у нас дерево.
2. Заполнять его теперь нужно строками для чего и переписываем функцию рандомного заполенения.
3. Теперь вносим наши строки в список и считаем их длинны.
4. Выводим дерево и результат нашего подсчета.
5. Осталось протестировать программу.
